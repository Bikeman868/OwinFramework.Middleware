Welcome to the index page

This page refrences the style sheet with versioning

Here are some links you can try, what to expect and why

URL What Why
/ The index.html page The DefaultDocument middleware is configured here to display index.html when the request URL contains no path. This page references the style sheet with a version number, so the browser will only request this file once and any changes you make to this file will not be reflected in the browser.
/home.html The home.html page The home.html page has a reference to the style sheet that is not versioned. Note that the LESS compiler tells the Output Cache middleware to cache and reuse the CSS output so that the compilation does not occur every time. This causes the output cache to add a header telling the browser to cache the file for a few hours. If the output cache gets a request for the CSS file and the underlying LESS file has been modified then it will not use the cached data, but pass the request on to the rest of the OWIN pipeline and the Less Middleware will re-compile the changed file.
/FormId/signup.html A form that you can use to create an account The Form Identification middleware is configured to handle POSTs to this URL. The page is static HTML containing a form with a submit button.
/FormId/signin.html A form that you can use to login The Form Identification middleware is configured to handle POSTs to this URL. The page is static HTML containing a form with a submit button.
/FormId/signout.html A form that you can use to log out The Form Identification middleware is configured to handle POSTs to this URL. The page is static HTML containing a form with a submit button.
/FormId/changePassword.html A form that you can use to change your password The Form Identification middleware is configured to handle POSTs to this URL. The page is static HTML containing a form with a submit button.
/owin/pipeline Visual representation of the OWIN pipeline The RouteVisualizer middleware is configured here to display this visualization at this URL.
/owin/endpoints A list of the URLs that work on this site The Documenter middleware is configured here to display this information at this URL. The Documenter middleware can generate documentation for any middleware that implements the ISelfDocumenting interface.
/ui A User Interface written in Dart The Dart middleware will detect the browser capabilities and rewrite requests to either the original Dart code or the compiled version. After passing through the Dart middleware, the request is handled by the Static Files middleware which serves the file. The Dart middleware configuration is here, and the static files middleware configuration is here.
/ui/favicon.ico The browser icon used for the Dart UI The Dart middleware will detect the browser capabilities and rewrite requests to either /ui/web/favicon.ico or /ui/build/web/favicon.ico
/ui/favicon_v1.ico The browser icon used for the Dart UI The Versioning middleware will strip the version number out of the request before the Dart middleware sees it, so this will work identically to /ui/favicon.ico.
/owin/analytics Analytic information from configured middleware The AnalysisReporter middleware is configured here to return analysis reports at this URL. The reporter gathers analtyics from any middleware that implements the IAnalysable interface.
/main.css The contents of main.less compiled into css The Less middleware is configured here to compile LESS into CSS on the fly. The resulting CSS will be stored in the Output Cache middleware so that it is not compiled again on each request.
/main_v1.css The contents of main.less compiled into css The Versioning middleware is configured here to strip version numbers off incomming requests. It is also configured to insert version numbers into URLs that refer to static assets, and add headers to the response telling the browser that it can cache these for a very long time.
/nonsense.html A message indicating that the URL is invalid The NotFound middleware is configured here to return a 404 response for all requests that were not handled earlier in the pipeline.
